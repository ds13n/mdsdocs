# Spacing

Mavka uses a consistent spacing system to create balance and rhythm across all interfaces. Everything is built on a **4px base grid**.

## Spacing Tokens

Use these tokens for margins, padding, and gaps.

| Token | Value | usage |
| :--- | :--- | :--- |
| `xs` | 4px | Micro-spacing, tight gaps |
| `sm` | 8px | Small components, secondary padding |
| `md` | 16px | Standard padding, component spacing |
| `lg` | 24px | Section gaps, major padding |
| `xl` | 32px | Large sections, hero spacing |
| `2xl` | 48px | Page-level margins |

## Grid System

We recommend the following container widths for responsive layouts:

- **Mobile**: Full width with `16px` side margins.
- **Tablet**: Optimized for `768px`.
- **Desktop**: Maximum container width of `1280px`.

## Best Practices

1. **Be Consistent**: Always use a token rather than hardcoded pixel values.
2. **Space for Breathing**: Don't be afraid of whitespace; it helps reduce cognitive load.
3. **Hierarchy**: Use larger spacing to separate unrelated sections and smaller spacing for related content.
