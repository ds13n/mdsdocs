# Mavka Designer Agent

The **Mavka Designer** is a specialized agentic assistant designed to build, maintain, and audit UI components following the Mavka Design System (MDS) standards. It bridges the gap between design tokens and implementation code.

## Core Capabilities

<CardGroup cols={2}>
  <Card title="Token Translation" icon="code">
    Automatically maps design intent to MDS tokens (colors, spacing, typography).
  </Card>
  <Card title="UI Auditing" icon="magnifying-glass">
    Reviews existing code for deviations from design standards and suggests fixes.
  </Card>
  <Card title="Accessibility" icon="universal-access">
    Ensures every generated component exceeds WCAG AAA standards by default.
  </Card>
  <Card title="Component Creation" icon="plus">
    Generates production-ready SwiftUI or React components in seconds.
  </Card>
</CardGroup>

## Interaction Principles

When collaborating with the Mavka Designer, the following principles apply:

1. **Design Intent Over Specifics**: Describe *what* you want to achieve (e.g., "a prominent call to action"), and the agent will determine the appropriate *Mavka* tokens.
2. **Context Awareness**: The agent understands the global `docs.json` configuration and ensures all new contributions fit the current project theme.
3. **Continuous Refinement**: The agent learns from feedback to improve its adherence to your specific brand's micro-adjustments within the Mavka framework.

## Example Workflow

> "Create a dashboard card with a title, a large numeric statistic, and a secondary action button."

**Mavka Designer Output:**
- Applies `Raised` elevation.
- Uses `Emerald 500` for the interaction token.
- Sets padding to `md (16px)` and title weight to `Semibold`.
- Ensures contrast ratios are > 7:1 for all text elements.
